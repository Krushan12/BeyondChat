## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/intercom-clone.git
   cd intercom-clone
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open http://localhost:3000 to view it in your browser.
